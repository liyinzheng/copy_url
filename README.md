
 ## 说明

 本项目为一个Chrome扩展，实现了如下功能：
 
 当页面为本地发布时（ip为 localhost 或者是 127.0.0.1），若要将链接分享给局域网内其他用户，需要将本地ip转换成局域网内的ip，然后路径部分采用当前路径。
 
 该扩展使用户只用点击一下按钮，就可以将转换后的ip放入剪贴板，直接复制即可。
